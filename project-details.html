Nix Project - Shoe E-Commerce Website
Nix is a fully functional shoe-based e-commerce website developed using React.js. It features a modern, responsive UI with seamless navigation, product listings, cart management, and a secure checkout process. The project integrates key functionalities like product filtering, user authentication, and a dynamic shopping experience, making it a complete and efficient online store for footwear enthusiasts.
